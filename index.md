## ⚙️ How it Works

Nova-Wizard acts as a bridge. It connects your computer to the Cloudflare platform. It writes the necessary code for a Cloudflare Worker. It pushes that code to your account. This setup process normally requires command line tools. Nova-Wizard replaces those tools with a visual interface. 

You do not need to install complex software like Node.js or Git. The app contains every component it needs to function. 

## 🛡 Security and Privacy

You keep your keys private. The app never sends your login details to any outside server. It creates a local connection between your desktop and Cloudflare. This ensures that only you access your proxy.

Because the proxy exists within your own Cloudflare account, you manage the traffic limits and the speed. Cloudflare provides a high level of security for the worker. 

## ❓ Frequently Asked Questions

### Does this cost money?
The free tier of Cloudflare works perfectly with this tool. Most users never hit the limits of the free plan.

### Can I run this on multiple computers?
Yes. You can install the tool on any number of Windows machines.

### What if I forget my password?
Nova-Wizard does not use passwords. It uses your OAuth session. This is a secure sign-in method that Cloudflare provides.

### I see an error message. What do I do?
Close the app and restart it. Check your internet connection. Ensure you remain logged into your Cloudflare account in your web browser.

### Can I delete the proxy later?
Yes. Simply log into your Cloudflare dashboard, go to Workers, and delete the script the app created.
